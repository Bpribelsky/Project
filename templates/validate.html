Validation Tools

The Validator is a web service that you use to make sure that all of the HTML code is valid according to the HTML5 specification.

You are required to run this tool on all URLs that you submit to Canvas. This validator will show you all of the mistake that are in your HTML code. Fix them and try again.

Submit your code to the HTML Validator. Copy and paste your URL.
